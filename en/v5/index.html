Vuesion

Vuesion is a boilerplate that helps product teams build faster than ever with fewer headaches and modern best practices across engineering & design.

Get Started →

Quick scaffolding

Create components, pages, vuex-modules, stories, interfaces, fixtures - and their tests - right from the CLI

Server-side-rendering

Use server-side-rendering, create a single-page-app or generate static HTML powered by Nuxt.js

Industry-standard i18n support

Scalable apps need to support multiple languages, easily add and support multiple languages

Progressive-Web-App

Offline-first support with service-workers and a lighthouse score as high as possible

Customizable Design System

Includes a blueprint for a 100% customizable, themeable Design System with 35+ components

Storybook

Allows interactive development, testing and sharing of UI components in various property states

Figma UIKit

The included Figma design system file includes the basics and all components of the Vuesion design system and it's perfectly in sync with storybook

GraphQL

includes a blueprint how to use vue-apollo with the auth-module, axios-module (re-authentication) and the composition-api

DX, Code quality and maintenance

TypeScript, Eslint, commit message lint, prettier and husky set up will help you to keep a high code quality
