Vuesion

Vuesion은 설계 전반에 사용할 수 있는 최신 사례들을 활용하여, 팀이 고민거리 없이 신속하게 서비스를 구축할 수 있도록 돕는 VueJS 보일러 플레이트입니다.

시작하기 →

빠른 프로토타입

CLI를 통해 컴포넌트, 페이지, Vuex 모듈, Storybook, 인터페이스 등을 만들고 테스트할 수 있습니다.

서버 사이드 렌더링

Nuxt.js를 활용해 서버 사이드 렌더링, 단일 페이지 앱 또는 정적 HTML 파일 생성을 빠르고 효율적으로 처리합니다.

업계 표준 i18n 지원

다양한 언어로 쉽게 번역할 수 있는 효율적인 프로젝트를 개발할 수 있습니다.

프로그레시브 웹 앱

서비스 워커를 적용하여 오프라인에서도 사용 가능한 앱을 제작하고, Lighthouse 점수를 상승시키세요.

유연한 디자인 시스템

35개 이상의 100% 커스터마이징 가능한 범용적인 디자인 컴포넌트를 포함합니다.

Storybook

다양한 상태의 UI 컴포넌트를 간편하게 구성하고, 테스트하며, 결과물을 자유롭게 공유할 수 있습니다.

Figma UI 키트

Storybook의 컴포넌트들과 완전히 일치하는 모든 컴포넌트를 포함한 Figma UI 키트를 제공합니다.

GraphQL

vue-apollo, auth-module, axios-module(재인증), composition-api 등을 활용한 코드 예제를 제공합니다.

DX, 코드 품질, 유지보수

Typescript와 Vuesion에 포함된 Eslint, 커밋 메시지 린트, prettier 등은 높은 코드 품질을 유지하는데 도움을 줍니다.
