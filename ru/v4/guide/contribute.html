# Поддержка проекта

Если Вам нравится этот проект, вы можете поддержать его:

  • Присоединяйтесь к нашему Discord-каналу (opens new window).

  • Помогите в популяризации Vuesion (opens new window)

  • Для написания кода и создания PRs, ознакомтесь с contribution guidelines (opens new window)

  • Принимаем любые идеи и предложения issue (opens new window)

  • Так же можно помочь с переводом и дополнением документации (opens new window)
