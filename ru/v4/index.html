Vuesion

Vuesion - стартовая сборка Vue JS для быстрого прототипирования с готовой архитектурой для больших приложений

Начало работы →

Быстрое прототипирование

Создавайте компоненты, страницы, vuex-модули, сторисы в Storybook, интерфейсы, фикстуры, а так же тестируйте их прямо из CLI

Server-side-rendering (SSR)

Генерации страниц на сервере (server-side-rendering), создание одностраничных (single-page-app) или генерация статических HTML-страниц обеспечивается Nuxt.js

Интернационализация i18n

Разрабатывайте приложения с поддержкой перевода на различные языки.

Progressive-Web-App

Разрабатывайте приложения, работающие офлайн с помощью service-workers и получайте максимальную оценку lighthouse

Гибкая дизайн-система

35+ готовых легко кастомизируемых UI-компонентов

Storybook

Разрабатывайте, тестируйте и делитесь с вашей командой UI-компонентами

Figma UIKit

Набор UI элементов Figma включает все основные элементы полностью соответствующие компонентам в Storybook

GraphQL

Используйте примеры кода для работы с vue-apollo и auth-module, axios-module (re-authentication) и composition-api

DX, Качество кода и его поддержка

TypeScript, Eslint, commit message lint, prettier и настроенный модуль husky обеспечит высокое качество кода
